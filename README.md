# UMKM Ledger Pro (React)

Versi React modern dari aplikasi ledger UMKM.

## Fitur
- Page 1: dashboard harian, quick entry, ledger hari ini
- Page 2: history transaksi penuh, filter lengkap, edit/delete, export Excel
- Page 3: inventory manager, add/edit/delete/restore, search/filter, export Excel

## Jalankan
Buka `index.html` lewat web server atau host ke GitHub Pages.

## Catatan
Aplikasi ini memakai React, Babel, dan SheetJS melalui CDN agar bisa langsung berjalan tanpa build step.
